About

Fix Your City is a student group project for Web-based application.

On the website registered users submit, vote and discuss problems in their cities, with variety of other accessible options.


Utilising MVC architecture with Laravel 5.1 framework (PHP) on backend to manage database (phpmyadmin, MariaDB) and security,
together AngularJS (1.4.8) on client-side to handle all content with a responsive design as almost all website content is dynamic.

Source code

Laravel library for authentification
AngularJS directive for image upload
AngularJS directive for image lightbox
CSS: Bootstrap 3.3.5
